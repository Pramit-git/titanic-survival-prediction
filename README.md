# titanic-survival-prediction
This project demonstrates the application of machine learning techniques to a real-world classification problem using the Titanic dataset. By performing data preprocessing and implementing models such as Random Forest and GradientBoost, meaningful patterns influencing passenger survival were identified.
### 📊 Results
- Accuracy: **90% using GradientBoost**

### 🛠️ Tech Stack
- Python, Pandas, Scikit-learn, GradientBoost
